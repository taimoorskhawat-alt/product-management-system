About ProductMS

A modern full-stack product management system built for learning and real-world usage.

🚀 What is ProductMS?

ProductMS is a full-stack web application that allows users to manage products efficiently with secure authentication and role-based access control.

✨ Key Features
  • JWT Authentication & Authorization
  • User Registration & Login System
  • Role-Based Access (Admin / User)
  • Full CRUD Operations on Products
  • Secure ASP.NET Core Web API
  • Angular 21 Frontend Integration
🛠 Tech Stack
  • Angular 21 (Frontend)
  • ASP.NET Core Web API (Backend)
  • SQL Server (Database)
  • Bootstrap 5 (UI Design)

Built as a full-stack portfolio project for learning modern web development.
